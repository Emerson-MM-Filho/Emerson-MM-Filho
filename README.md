# Hi! I'm Emerson!

I started programming while studying Architecture, I was looking for a way to reduce rework in project design and I found the solution in programming. At a certain point I realized that I was identifying myself more as a developer than an architect. So, in 2020, I started my career change, I started the [@Trybe](https://github.com/tryber) course, an extensive web programming course, which lasts one year and is 100% online and live classes. Within 6 months of course I got my first job opportunity in the programming area, at [@Deucredito](https://github.com/DEUcredito), as a full stack with a focus on back-end, using Python and Django.
